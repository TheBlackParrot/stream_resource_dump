Truculenta Variable Font
========================

This download contains Truculenta as both a variable font and static fonts.

Truculenta is a variable font with these axes:
  opsz
  wdth
  wght

This means all the styles are contained in a single file:
  Truculenta-VariableFont_opsz,wdth,wght.ttf

Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
