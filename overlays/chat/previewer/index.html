This page allows you to customize your chat appearance on streams that use my chat overlay! Changes may not apply if the streamer disables the ability for you to change these settings, this is up to the streamer to allow them or not.

Streamers can also ensure custom colors are between set brightness thresholds and font sizes will scale relative to the streamer's set default font size, so what you see here may not quite reflect what actually appears on the overlay.

Changes do not save automatically, you must manually press the Save button in each section. If you want to go back to your previously saved settings, refresh the page.
