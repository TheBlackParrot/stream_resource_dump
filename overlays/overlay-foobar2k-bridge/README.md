**This is a Node.js bridge script that bridges foobar2000 and my Now Playing streaming overlay.**

## Installation
1. Install [Node.js](https://nodejs.org/en)
2. Run `npm i` in the same folder as the code to install package dependencies
3. Start foobar2000
4. Run `node main.js`

Requires the [foo_beefweb component](https://www.foobar2000.org/components/view/foo_beefweb)
